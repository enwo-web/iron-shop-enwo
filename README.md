# 🛒 درباره پروژه

**Iron Shop** یک نمونه‌کار Front-End از یک فروشگاه اینترنتی مدرن برای نمایش و فروش اکسسوری های ترند است. تمرکز این پروژه بر طراحی رابط کاربری حرفه‌ای، تجربه کاربری روان و نمایش جذاب محصولات بوده است.

> ⚡ این پروژه صرفاً جهت نمایش مهارت‌های طراحی رابط کاربری و توسعه Front-End ساخته شده و به هیچ سیستم پرداخت یا دیتابیس متصل نیست.
